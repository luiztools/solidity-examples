# NFT ERC-721 HardHat
A NFT contract example in Solidity with HardHat (ERC-721).

## Pre-requisites

1. Node.js

## How to Run

1. open command line terminal
2. cd nft-erc721-hardhat
3. npm install
4. npx hardhat test
