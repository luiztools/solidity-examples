# token-erc20-hardhat

A simple token written in Solidity using ERC-20 spec, OpenZeppelin contracts and HardHat toolkit.

## Pre-requisites

1. Node.js

## How to Run

1. open command line terminal
2. cd token-erc20-hardhat
3. npm install
4. npx hardhat test
