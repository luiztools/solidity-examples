# crud-solidity-foundry
A CRUD example in Solidity with Foundry.

## Pre-requisites

1. Foundry

## How to Run

1. open command line terminal
2. cd crud-solidity-foundry
3. forge build
4. forge test

## How to Deploy
1. copy .env.example as .env
2. fill .env variables
3. review hardhat.config.ts
4. npm run deploy or npm run ignition
