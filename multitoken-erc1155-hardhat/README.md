# multitoken-erc1155-hardhat

A simple multi-token contract written in Solidity using ERC-1155 spec, OpenZeppelin contracts and HardHat toolkit.

## Pre-requisites

1. Node.js

## How to Run

1. open command line terminal
2. cd multitoken-erc1155-hardhat
3. npm install
4. npx hardhat test
